**Hospital Emergency Room Dashboard | Microsoft Excel**
**Project Overview**
The Hospital Emergency Room Dashboard is an end-to-end data analytics project developed in Microsoft Excel to analyze Emergency Room (ER) operations and provide actionable insights for healthcare stakeholders.
This interactive dashboard enables users to monitor key hospital performance metrics, evaluate patient flow, analyze waiting times, assess patient satisfaction, and identify operational bottlenecks. By transforming raw healthcare data into meaningful visualizations, the dashboard supports informed decision-making and improves the overall efficiency of emergency room management.

**Business Problem**
Hospitals generate a large volume of patient data every day, making it difficult to monitor operational performance through raw records alone. This project addresses that challenge by creating an interactive dashboard that helps hospital administrators:
- Monitor daily ER performance
- Track patient admission trends
- Analyze patient waiting times
- Measure patient satisfaction
- Understand patient demographics
- Evaluate departmental referrals
- Improve operational efficiency through data-driven insights

**Project Objectives**
- Analyze Emergency Room patient data using Microsoft Excel.
- Build an interactive dashboard for hospital management.
- Track key healthcare KPIs in real time.
- Identify trends in patient admissions and waiting times.
- Analyze patient demographics and referral patterns.
- Enable users to filter insights by month and year.

**Key Performance Indicators (KPIs)**
The dashboard tracks the following KPIs:
- Total Number of Patients
- Average Patient Waiting Time
- Patient Satisfaction Score
- Admission vs Non-Admission Rate
- Patient Attended Status (On-Time vs Delay)
- Patient Distribution by Gender
- Patient Distribution by Age Group
- Department Referral Analysis

**Dashboard Features**
The dashboard provides interactive analysis for:
- Monthly Emergency Room Performance
- Year-wise Comparison (2023 & 2024)
- Total Patients Visited
- Average Waiting Time
- Patient Satisfaction Score
- Admission Status Analysis
- On-Time vs Delayed Patient Attendance
- Gender-wise Patient Distribution
- Age Group Analysis
- Department Referral Trends
- Interactive Month and Year Filters

**Key Insights**
- Monitor total ER patient visits and identify high-volume periods.
- Compare admitted and non-admitted patient percentages.
- Evaluate average waiting time to improve operational efficiency.
- Measure patient satisfaction as a key service quality indicator.
- Analyze patient demographics by age group and gender.
- Identify departments receiving the highest number of referrals.
- Track on-time versus delayed patient attendance to assess service performance.
